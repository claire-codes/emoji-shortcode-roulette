Emoji Shortcode Roulette

::
What is this?
Generate a randomly selected emoji shortcode and its GitHub image each time the button is pressed. Emoji Shortcodes are codes used on various websites to speed up emoji insertion using a keyboard (Emojipedia). The emoji data is from the GitHub emoji api https://api.github.com/emojis, although the codes will mostly work in other apps and websites.
